methods.list (lookup_type, value, need_parent_list)
It returns a complete list of Lookup values corresponding to the specified "type". If the "value" parameter is specified, only the related heading is returned. If “parentList” takes the “True” value, it returns the complete hierarchy available for the multilevel Lookup lists.

Lookup:create (lookup_type, code, description, id, notes, parent_id, position)
Lookup:create It creates in the database a new heading of a data Lookup list containing information inserted in the “Lookup” object. It returns the "id" identification attribute.

Parameters:

  • lookup_type - Name of the Lookup list which includes the current heading (string)
  • code - Code of the Lookup heading (one single heading of a Lookup list).(string)
  • description - Description of the Lookup heading (one single heading of a Lookup list) (string)
  • id - Lookup identification, it is automatically assigned by the database (number)
  • notes - Notes connected with the Lookup heading (string)
  • parent_id - Identification of the parent Lookup in the current heading (if applicable) (number)
  • position - Location of the Lookup heading in the related Lookup list (number)

Returns:

    id (integer)
